What does 'nullref' mean?

A reference, as defined by Wikipedia, is a value that enables a program to indirectly access a particular datum, such as a variable or a record,
in the computer's memory or in some other storage device. Thus, a null reference (or nullref) is a reference that points to nothing, as the name implies.
